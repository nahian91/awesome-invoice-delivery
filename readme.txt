Generate professional Invoices, Packing Slips, and Delivery Notes for WooCommerce orders directly from your WordPress admin.

== Description ==

Awesome Invoice, Delivery Notes & Packing Slips allows WooCommerce store owners to easily generate and print professional order documents.

**Features:**
* Generate Invoice, Packing Slip, and Delivery Note for each order.
* Fully customizable company information: name, address, email, phone, and logo.
* Print-ready HTML documents.
* Secure with WordPress nonces and proper escaping.
* Lightweight and fast; no external dependencies.
* Compatible with the latest WooCommerce and WordPress 6.6+.

== Installation ==

1. Upload the `awesome-invoice-delivery` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Invoices > Settings** in the WordPress admin to configure your company details.
4. Go to **Invoices** to view orders and generate documents.

== Frequently Asked Questions ==

= Can I generate documents for multiple orders at once? =

Currently, documents are generated one order at a time. Bulk generation may be added in future updates.

= Can I customize the look of the documents? =

Yes, you can modify the plugin's HTML templates or override them via your theme.

= Does it support PDF generation? =

No, this version generates HTML documents for printing. PDF export may be added in future releases.
